Financial Calculator

Instructions:

    1) Enter the principal amount in dollars.

    2) Input the annual interest rate as a percentage.

    3) Specify the time duration in years.

    4) Choose the number of compounding periods per year.

    5) If calculating annuities, enter the payment amount.

    6) Provide an annual growth rate for payments if applicable.

    7) If dealing with deferred annuities, specify the deferred period.

    8) Select the calculation type from the dropdown menu.

    9) Click the "Calculate" button to get results.
